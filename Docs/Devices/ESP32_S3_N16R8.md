## Teknik Özellikler

### İşlemci
| Parametre | Değer |
|-----------|-------|
| CPU | Xtensa 32-bit LX7 dual-core |
| Frekans | 240 MHz |
| İç SRAM | 512 KB |
| ROM | 384 KB |

### Bellek
| Parametre | Değer |
|-----------|-------|
| Flash | 16 MB (N16) |
| PSRAM | 8 MB (R8) |

### Haberleşme
| Protokol | Detay |
|----------|-------|
| Wi-Fi | 2.4 GHz, 802.11 b/g/n, 150 Mbps |
| Bluetooth | 5.0 LE + Mesh |
| SPI | 4× |
| UART | 3× |
| I²C | 2× |
| I²S | 2× |
| USB | OTG + JTAG (dual Type-C) |

### GPIO & Analog
| Parametre | Değer |
|-----------|-------|
| GPIO | 45 programlanabilir pin |
| ADC | 2× 12-bit, 20 kanal |
| Touch | 14× kapasitif GPIO |
| LCD | 8-16 bit RGB arayüz |
| Kamera | DVP arayüz |

### Güvenlik
- AES şifreleme (donanım)
- RSA, HMAC
- WS2812 RGB LED desteği

---

## Proje Notları

- Ana kontrol birimi — tüm sensörler ve ESC sinyalleri buraya bağlanır
- WiFi → uzaktan telemetri veya web arayüzü
- Bluetooth → yakın mesafe RC kumanda
- 16MB Flash: firmware + waypoint haritası için yeterli
- 8MB PSRAM: sensör tamponu, log için yeterli
- ADC ile batarya voltajı okunabilir (gerilim bölücü üzerinden)
- I²C → IMU, GPS modülleri
- UART → GPS NMEA

### Pin Planı (Öneri)
| Fonksiyon | Pin Tipi |
|-----------|----------|
| ESC PWM (×2) | GPIO (dijital çıkış) |
| Hall Effect (PDB) | ADC veya UART |
| Voltaj sensör (PDB) | ADC |
| GPS | UART |
| IMU | I²C |
| RC alıcı | UART veya PWM giriş |

- Su geçirmez muhafazaya alınmalı
