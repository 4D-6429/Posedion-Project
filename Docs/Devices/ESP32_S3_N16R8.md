| UART | 3× |

## Proje Notları

- Ana kontrol birimi — tüm sensörler ve ESC sinyalleri buraya bağlanır
- WiFi → uzaktan telemetri veya web arayüzü
- Bluetooth → yakın mesafe RC kumanda
- 16 MB Flash: firmware + waypoint haritası için yeterli
- 8 MB PSRAM: sensör tamponu ve log için yeterli
- ADC ile batarya voltajı okunabilir (gerilim bölücü üzerinden)
- I²C → IMU
- UART → GPS (NMEA), RC alıcı (iBUS)

### Pin Planı
| Fonksiyon | Pin | Protokol |
|-----------|-----|----------|
| ESC PWM Sol | GPIO17 | MCPWM çıkış |
| ESC PWM Sağ | GPIO18 | MCPWM çıkış |
| RC alıcı (iBUS) | GPIO16 | UART1 RX |
| GPS | TBD | UART2 |
| IMU | TBD | I²C |
| Voltaj sensörü (PDB) | TBD | ADC |
| Hall Effect (PDB) | TBD | ADC veya UART |
